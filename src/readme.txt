== Installation ==

Using this plugin is very simple. All you have to do is:

1. Upload the `render` folder to the `/wp-content/plugins/` directory

2. Activate the plugin through the 'Plugins' menu in WordPress

3. View available shortcodes by going to Shortcodes in the admin menu
